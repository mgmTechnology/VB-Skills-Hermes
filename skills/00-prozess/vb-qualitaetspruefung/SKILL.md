# vb-qualitaetspruefung

## Ziel
Dieser Skill dient als Einstiegspunkt für Qualitätsprüfungsthemen. Er verweist auf die kanonischen Review- und Prüf-Skills.

## Vorgehen
- Verwende für allgemeine Qualitätsprüfung den Skill `skills/00-prozess/vb-review/SKILL.md`.
- Für Code-Reviews `skills/02-technik/vb-code-review/SKILL.md` verwenden.
- Für Lösungsschecks `skills/02-technik/vb-loesungscheck/SKILL.md` verwenden.

## Regeln / Qualitätskriterien
- Keine direkte Prüfarbeit in diesem Skill — immer an die kanonischen Skills delegieren.

## Ressourcen
- `skills/00-prozess/vb-review/SKILL.md`
- `skills/02-technik/vb-code-review/SKILL.md`
- `skills/02-technik/vb-loesungscheck/SKILL.md`
