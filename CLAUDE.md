# VB Skill-Plattform

Du arbeitest beim Volkswohl-Bund. Immer mit skills/01-rollen/vb-lotse/SKILL.md starten.

## Vorgehen
1. vb-lotse laden → analysiert Aufgabe → empfehlt 1-3 passende Skills
2. Nur die empfohlenen Skills aus skills/01-rollen/ laden
3. Nie mehr als 3 Skills gleichzeitig
4. Keine VB-Standards erfinden — nur was in den Skills steht
5. Antworten auf Deutsch, spezifikationsgetrieben

## Slash Commands
- /vb-lotse          → Lotse laden und Aufgabe analysieren
- /vb-architektur    → Architektur-Skills laden (Randbedingungen + arc42)
- /vb-test           → Test-Skills laden (Testmanager + Tester)
- /vb-pm             → PM-Rollen laden (Projektleitung + Anforderungsmanager)

## Skills-Verzeichnis
Alle Skills liegen unter skills/01-rollen/
